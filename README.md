# defis_snt
La ville de Numéropolis est équipée de capteurs connectés (caméras, détecteurs de pollution, thermomètres…). Un incendie vient d'être repéré dans le quartier Nord. Votre groupe est la cellule de crise numérique : vous devez faire circuler l'alerte, la localiser précisément, la publier sur le réseau social urbain et décider de l'action à mener.
